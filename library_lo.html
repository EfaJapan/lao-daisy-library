⬇ ទាញយកសៀវភៅ! / Download the Book!

👆 ចុចប៊ូតុង "Download E-pub"
Tap the "Download E-pub" button on this page
↓
📱 ឯកសារត្រូវបានរក្សាទុកក្នុងឧបករណ៍
The .epub file is saved to your device
💡 iPhone ប្រើ Safari: ចុច ⬇ នៅរបារ browser → ចុចឈ្មោះឯកសារដើម្បីរក្សាទុក
On iPhone with Safari: tap ⬇ icon in browser bar → tap filename to save
3

📂 នាំ EPUB ចូល EasyReader / Add EPUB to EasyReader

▼ ជំហានទី 1 — រកឯកសារ EPUB / Find the EPUB file
📱 ① បើក App Files (ឯកសារ) 📁 ក្នុង iPhone
① Open the Files app on your iPhone
↓
🔍 ② ចុច "Browse" (ក្រោមស្ដាំ) → "On My iPhone" → "Downloads"
② Tap "Browse" (bottom right) → "On My iPhone" → "Downloads"
↓
👆 ③ រក ហើយចុចឯកសារ .epub
③ Find and tap the .epub file
▼ ជំហានទី 2 — បើកក្នុង EasyReader / Open in EasyReader
📤 ④ ចុចរូបតំណាង Share (□↑) នៅក្រោមអេក្រង់
④ Tap the Share icon (□↑) at the bottom of the screen
↓
🔽 ⑤ រំកិលចុះ → ចុច "Copy to EasyReader"
⑤ Scroll down → tap "Copy to EasyReader"
↓
✅ ⑥ EasyReader បើក → ចុច "My Books" → អានបាននៅហើយ!
⑥ EasyReader opens → tap "My Books" → start reading!
💡 ប្រសិនបើរក "Copy to EasyReader" មិនឃើញ:
រំកិលស្ដាំ → ចុច "More" → ធ្វើឱ្យ EasyReader ដំណើរការ
Can't find "Copy to EasyReader"? Scroll right in Share list → tap "More" → enable EasyReader

💡 ប្រសិនបើឯកសារមិននៅក្នុង Downloads:
ព្យាយាម Files → iCloud Drive → Downloads ឬ 🔍 ស្វែងរក ".epub"
Try Files → iCloud Drive → Downloads or Search 🔍 for ".epub"
▼ ជំហានទី 1 — រកឯកសារ EPUB / Find the EPUB file
📁 ① បើក App Files / File Manager
① Open the Files or File Manager app
↓
📥 ② ចូលថត Downloads → រកឯកសារ .epub
② Open Downloads folder → find the .epub file
↓
👆 ③ ចុចឯកសារ .epub
③ Tap the .epub file
▼ ជំហានទី 2 — បើកក្នុង EasyReader / Open in EasyReader
📋 ④ អេក្រង់ "Open with" លេចឡើង → ជ្រើស EasyReader
④ "Open with" screen appears → select EasyReader
↓
✅ ⑤ EasyReader បើក → ចុច "My Books" → អានបាននៅហើយ!
⑤ EasyReader opens → tap "My Books" → start reading!
💡 ប្រសិនបើ EasyReader មិនមាននៅក្នុងបញ្ជី:
បើក EasyReader មុន 1 ដង → ហើយព្យាយាមម្ដងទៀត
If EasyReader doesn't appear, open EasyReader once first → then try again
▼ ជំហានទី 1 — រកឯកសារ EPUB / Find the EPUB file
💻 ① បើក File Explorer → ចូលថត Downloads
① Open File Explorer → go to Downloads folder
↓
🔍 ② រកឯកសារ .epub
② Find the .epub file
▼ ជំហានទី 2 — បន្ថែមទៅ EasyReader / Add to EasyReader
📖 ③ បើក EasyReader → ចុច "Add Book" → ជ្រើសឯកសារ .epub
③ Open EasyReader → click "Add Book" → select the .epub file
↓
✅ ④ សៀវភៅនៅក្នុង My Books → អានបាននៅហើយ!
④ Book appears in My Books → start reading!
💡 អ្នកអាចអូស ឯកសារ .epub ពី File Explorer ចូល EasyReader ដោយផ្ទាល់!
You can also drag the .epub file from File Explorer directly into EasyReader!

📖 About This Library

🎯 គោលបំណង / Purpose

បណ្ណាល័យនេះដំណើរការក្រោមសន្ធិសញ្ញា Marrakesh ដើម្បីផ្ដល់ការចូលប្រើសៀវភៅឌីជីថល សម្រាប់កុមារដែលមានការលំបាកក្នុងការអាន។

This library operates under the Marrakesh Treaty to provide accessible digital books for children with reading difficulties, including those from disadvantaged backgrounds.

👶 អ្នកប្រើប្រាស់គោលដៅ / Target Users

កុមារគ្រប់រូបដែលត្រូវការការគាំទ្រក្នុងការអាន — រួមមានកុមារពិការ កុមារក្នុងគ្រួសារក្រីក្រ និងកុមារនៅតំបន់ឆ្ងាយ។

All children who need reading support — including children with disabilities, children from low-income families, and children in remote areas.
